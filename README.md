## Development
To start the development server run:
```bash
bun run dev
```

Open http://localhost:3000/ with your browser to see the result.

## Env
Copy `.env.example` and set values if needed.

Defaults:
- `LISTEN_IP=127.0.0.1`
- `PORT=3000`

## Testing
Tests live in `test/`.

```bash
bun test
```
